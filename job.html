Job Opportunity

A job that involves HTML/CSS/JavaScript

A great job you can get by learning HTML, CSS, and Javascript is a web developer.

What is a web developer?
A web developer uses their creative skills as well as their knowledge on CSS, HTML, and JavaScript to create amazing websites. They make wesites look nice using CSS, and use JavaScript to add effects and other features to it.
Why do I want to become a web developer?
I want to become a web developer because I think it would be fun to make websites for other people. I could help them make it look nice and work.
